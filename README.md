# ALU Machine Learning Year 2 Trimester 3 Formative 3 - Group 1

This repository contains the assignments completed for the African Leadership University (ALU) Mathematics for Machine Learning Course, Year 2, Trimester 3, Formative 3. The project focuses on applying concepts of Probability Distributions, Bayesian Probability, and implementing Gradient Descent from scratch.

## Project Structure

The project is divided into four main components:

1.  **Part 1: Probability Distributions** -

_by Hamse Jama_

    - Analyzed the 'Tips' dataset.
    - Built a custom Python implementation of the Probability Density Function (PDF) for a bivariate normal distribution from scratch.
    - Calculated PDF values and visualized the distribution using Matplotlib and Seaborn (Contour and 3D surface plots).

2.  **Part 2: Bayesian Probability** -

_by Divine Itu and Alliane Umutoniwase_

    - Analyzed the IMDb Movie Reviews Dataset.
    - Applied Naive Bayes principles for sentiment analysis by selecting specific keywords.
    - Handled categorical probabilities using Pandas and evaluated results mathematically.

3.  **Part 3: Handwritten Gradient Descent** -

_by Michael Nwuju, Hamse Jama, Divine Itu, and Alliane Umutoniwase_

    - Collaborative step-by-step mathematical proof of the forward pass, loss calculation (MSE), backpropagation (Chain Rule), and parameter update rules over 4 iterations.

4.  **Part 4: Manual Gradient Descent Code Implementation** -

_by Michael Nwuju_

    - Applied vectorization and broadcasting techniques to implement gradient descent for linear regression without utilizing high-level ML frameworks.
